## 🛠️ Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Fonts:** JetBrains Mono, Inter

## 🚀 Running the Portfolio

### Development Mode
```bash
cd Portfolio
npm install
npm run dev
```
Visit: http://localhost:3000

### Production Build
```bash
npm run build
npm start
```
