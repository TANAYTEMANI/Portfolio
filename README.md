# Tanay Agarwal - Portfolio

A minimalistic, terminal-inspired portfolio website showcasing my experience as an AI Engineer and Developer.

## Features

- 🎨 Terminal/code-inspired design aesthetic
- ⚡ Built with Next.js and Tailwind CSS
- 📱 Fully responsive design
- 🎭 Smooth animations and transitions
- 💼 Comprehensive sections: About, Experience, Projects, Skills, Education, Achievements, Contact
- 📄 Downloadable resume

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Font:** JetBrains Mono (monospace), Inter (sans-serif)

## Getting Started

### Prerequisites

- Node.js 16.x or higher
- npm or yarn

### Installation

1. Clone the repository or navigate to the Portfolio folder

```bash
cd Portfolio
```

2. Install dependencies

```bash
npm install
```

3. Run the development server

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```

## Customization

To customize the portfolio for your own use:

1. Update content in each component file
2. Modify colors in `tailwind.config.js`
3. Replace resume PDF in `public/` folder
4. Update contact information in `Contact.jsx`
5. Modify social links and personal details

## Design Inspiration

This portfolio takes design inspiration from terminal/code-based aesthetics with:
- Monospace typography
- Green accent colors (#00ff00)
- Dark background (#0a0a0a)
- Command-line style headers
